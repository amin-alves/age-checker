# Verificador de Idade

## Descrição
Programa simples que verifica se a pessoa é maior ou menor de idade.

## Funcionalidades
- Pergunta o nome do usuário
- Pergunta a idade do usuário
- Verifica se é maior ou menor de idade
- Mostra o resultado no terminal

## Tecnologias
- Python 3

## Como usar
1. Abra o terminal
2. Navegue até a pasta do projeto
3. Execute o arquivo:
```bash
python verificador_idade.py
